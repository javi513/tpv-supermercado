# 🛒 TPV Supermercado - Prácticas FP DAW

Sistema de Terminal de Punto de Venta para supermercado desarrollado como proyecto final de prácticas de **Desarrollo de Aplicaciones Web (DAW)**.

## Tecnologías utilizadas

- **Frontend**: React + TypeScript + Vite + TailwindCSS
- **Backend**: Node.js + Express + TypeScript
- **Base de datos**: Supabase (PostgreSQL)
- **ORM**: Prisma
- **Autenticación**: JWT + bcrypt

## Estructura del proyecto

tpv-supermercado/
├── frontend/          → Aplicación React (pantalla login + TPV)
├── backend/           → API con Node.js + TypeScript + Prisma
├── prisma/            → Schema de la base de datos
├── .env.example       → Plantilla de configuración
└── README.md
